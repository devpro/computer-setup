# Windows 11

## Display

### Screens

Review and update:

* The refresh rate for every screen.

> Settings (Windows) > System > Display > Advanced display (under Related settings) > Choose a refresh rate

* G-SYNC

> NVIDIA Control Panel > Display > Set up G-SYNC > Enable G-SYNC
